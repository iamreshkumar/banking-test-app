### This is sample Application - with just a basic API representations


## Banking Spring Boot Application

### Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- AOP (Audit Logging)
- Swagger
- JUnit 5

### Banking Features
- Customer onboarding
- Account management
- Credit / debit transactions
- Optimistic locking
- Pagination for transaction history
- DTO-based join queries
- Centralized exception handling
- Audit logging using AspectJ

### How to Run
1. mvn clean install
2. mvn spring-boot:run
3. Swagger: http://localhost:8080/swagger-ui.html
